# Credit Team Website

## Authors: Emily Wood

## Overview

The "Credit" website is designed to provide valuable information and resources to enhance users' understanding of their credit score and current loans. This website's front end has been mostly completed, and future tasks involve implementing the backend code and potential improvements to the front end.


## Features

- **Credit Meter:** Designed to give users information on their credit score. Information is conveyed using a number, colors, and a credit meter figure. The colors range from red to green, given different numbers and the credit meter dial should move depending on the credit score.

- **Loans:** Loan balance is shown in a table and in a graph. As a person's loan balances increase and decrease, this should be shown both in table and in the graph.

- **Financial Advice:** Financial advice should link to the "Financial Literacy" page to help a user become more acquainted with different methods of improving relevant financial topics.

## Technologies Used

- **HTML:** The structure of the website is built using HTML to ensure a clear and organized layout.

- **CSS:** Styling is implemented using CSS to create an aesthetically pleasing and user-friendly interface.
